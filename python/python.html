    the math module | import math

    Author : Dipayan Dutta | 10 Dec 2016

    From the system admin point of view may be math module is not looking so important now , but it will be make full sense to create this blog , because sometime we admins need to analys the log files and for that we need some operations , those operation can be performed with this module , doing this operation with python are super simple . And for the Data Science , This module is the most basic thing should start .

    Most Frequently used math methods in python

    First of all lets import all of the methods from the math module

    from    math  import   *

    Methods Application
    sqrt(16) Will Compute the square root of a number
    sqrt(2.3) It is also able to compute for floating numbers
    exp(2) Will Calculate e to the power 2 i.e. e^2
    log(12) To get the value of log 12
    log10(12) get the value of log 12 base 10
    pow(2,3) to calculate the value of 2 to 3 i.e. x^y
    sin(45) Get the value of sin 45 in radians
    cos(45) Get the value of cos 45 in radians
    tan(45) Get the value of tan 45 in radians
    degrees(1) Convert Radian values in degrees
    radians(57.59577) Convert Degree in to radian
    factorial(5) This will get the factorial of 5
    sum([1,2,3]) To get the sum of array , array must be contains all integer values
    fsum([1.3,3.4]) Similar working as sum function but one step ahead it will work on an array of floating numbers

    ★  There are also other methods present in this math module , here i only pointed out the most frequent one's .
