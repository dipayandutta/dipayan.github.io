    the platform module | import platform

    Author : Dipayan Dutta | 13 Dec 2016

    From the system admin point of view this one of the most important module , this module is emmensely useful to get the information about the underlying system, this module works on Linux , mac , windows . Here i am demonstrating the use and operations in the linux machine.

    Most Frequently used platform methods in python

    First of all lets import platform module

      import   platform

    Methods Application
    python_version() Get the running python version
    python_version_tuple() Returns a tuple on major , minor and patchlable on python version
    uname() similar as linux uname command
    system() Get the OS type
    release() get the kernel version
    platform.node() get the host name
    platform.architecture() Will show the system architecture , i.e. 64bit or 32 bit
    linux_distribution() Get the details of the linux distributor

    As some of the methods returns tuple as an output , we can extract the index for getting more specific informations , some of them are explained below .
